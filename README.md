# Software de newsletter - captura e gerenciamento De e-mails
### Stack
- PHP 8.1:latest
- Laravel 10
- Blade
- Tailwind CSS 3.3.3
- Apache
- MySQL 8
- Vite

- PSRs
- Clean Code

# Run
### Create database and set .env
```
create database newsletter;
```

### Js
```
npm install
npm run dev
```

### PHP 
```
composer install
php artisan migrate:refresh --seed
php artisan serve
```
